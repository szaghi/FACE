# About FACE

FACE (Fortran Ansi Colors and Styles Environment) is a KISS pure Fortran library for
easy *colorizing* and *stylizing* strings with ANSI escape codes — just one **function**.

- FACE is a pure Fortran (KISS) library
- FACE is Fortran 2008+ standard compliant
- FACE is tiny
- FACE is a Free, Open Source Project
